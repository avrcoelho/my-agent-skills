# Clean Code JS/TS Skill

This skill provides strict guidance for writing clean, maintainable, and professional JavaScript and TypeScript code. It is based on industry-standard principles adapted from common clean code practices.

## How to use this skill

1.  **Strict Adherence**: When this skill is active, you MUST strictly follow the rules documented in [AGENTS.md](./AGENTS.md).
2.  **Contextual Application**: Apply these rules to every line of code you generate, refactor, or review.
3.  **Examples as Benchmarks**: Use the "Correct" vs "Incorrect" examples in `AGENTS.md` as the standard for your implementations.

## Scope

- **Variables**: Naming, searchability, and context.
- **Functions**: Arguments, purity, single responsibility, and naming.
- **Objects and Classes**: Patterns, inheritance vs. composition, and method chaining.
- **SOLID**: Implementation of all five SOLID principles.
- **Asynchrony**: Proper use of Promises and Async/Await.
- **Error Handling**: Exception management and reporting.
- **Comments**: Self-documenting code and removal of dead code.

Refer to [AGENTS.md](./AGENTS.md) for the detailed specification and examples.
